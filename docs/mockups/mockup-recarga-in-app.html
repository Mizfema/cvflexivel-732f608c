Regras de exibição para a equipa frontend

Chip no header: visível apenas para donos de pacote avulso. Verde ≥4 créditos, amarelo 1–3, vermelho 0.
Banner inline: aparece no editor quando saldo cai a ≤3 créditos, uma vez por sessão. Dispensável (dismiss), reaparece só na sessão seguinte se o saldo continuar a ≤3.
Modal de esgotamento: só quando saldo=0 e o utilizador tenta uma acção premium. Nunca proactivamente. Oferece sempre as duas vias (recarga e upgrade para mensal) — nunca só a recarga.
Regra dourada: nada disto aparece na página /planos, em nenhum estado. A página /planos mostra 4 ofertas — grátis, avulso 149, mensal 349, trimestral 749.
